## Doc
### createContext
```
const I18nContext = createContext<Translations | null>(null);

This is a react function that creates a Context, a global storage for data.
Translations: your translation type
| null: or null
(null): default value is null
```
### children
```
export function I18nProvider({ children }: { children: React.ReactNode })
{ children }: take the children property out of that props object

<I18nProvider>
  <App />   {/* this is the children */}
</I18nProvider>
```
### Provider
```
return (
    <I18nContext.Provider value={translations}>
      {children}
    </I18nContext.Provider>
  );
  
This is the Provider from the React Context you created earlier, it's used to share data with all child components.

value={translations}: Every child component can read this value


how to use it:
const t = useContext(I18nContext);
```
